composite Main

This main composite demonstrates the usage of the HTTPParallelRequest operator. The sample logic sends a number different input tuples to HTTPRequestAsync composite operator and prints the received results

Parameters

  • url: A list of urls url to get
  • method: The list with the method identifiers
  • data: A list with data (to push or put)
  • iterations: The number of operations to perform
  • period: The period of operations (if bult is false)
  • burst: If set to true all requests are sent in a burst
  • noConn: Number of connections
